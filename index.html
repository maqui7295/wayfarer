      WayFarer

      WayFarer is a public bus transportation booking server. You can also view this documentation over at Wayfarer.

      Resource Group ¶

      Way Farer API Root  ¶

      This resource does not have any attributes. Instead it offers the initial API affordances in the form of the links in the JSON body.

      It is recommend to follow the “url” link values, Link or Location headers where applicable to retrieve resources. Instead of constructing your own URLs, to keep your client decoupled from implementation details.

      Authentication ¶

      Resources related to authentication of users of the API.

      User registration  ¶

      create a user
      POST/auth/signup

      You may create users using this action. It takes a JSON object containing the credentials below.

      • email (string) - The email of the new user

      • password (string) - The password of the new user

      • first_name (string) - The first name of the new user

      • last_name (string) - The last name of the new user

      Example URI

      POST https://mark-wayfarer.herokuapp.com//auth/signup
